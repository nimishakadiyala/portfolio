ABOUT ME

I am a passionate web developer with a strong background in Java programming and front-end development. During my undergraduate studies, I gained hands-on experience in web development, which sparked my enthusiasm for designing and developing webpages and websites.

At IBM, I had the opportunity to further enhance my skills as a web developer. Working on projects like RAD, I improved code analysis and debugging tools while ensuring compatibility with the latest Java frameworks and technologies. I became proficient in HTML5, JavaScript, ReactJS, CSS, XML, SCSS, AJAX controls, and JSON, actively participating in the software development life cycle. Continuing my education, I pursued a master's degree at California State University, Fresno. As a web developer student assistant, I collaborated on projects using Java, HTML, and ReactJS. I developed single-page applications, adhering to accessibility and web browser standards, and supervised a CMS project utilizing OMNI CMS.

At Wichita State University, I worked as a web developer student assistant for Counseling and Prevention Services (CAPS). I specialized in developing custom WordPress themes and templates using HTML, CSS, and JavaScript. I collaborated with designers and backend developers to seamlessly integrate frontend interfaces and functionalities within WordPress plugins. Additionally, I gained exposure to Angular, expanding my knowledge of the framework.
